# Summer Chores
- **Description**: A javascript file that follows someone working through a list of chores.
- **Tool Requirements**: Requires Node.js
- **Starting Program**: Simply:
    1. In an appropriately scoped terminal (either navigated to the project folder, or an intigrated terminal in the IDE), run the `node [filename]` command for the intended version.
        1. For the "Callback"/"callback hell" version `node callbackVersion.js`
        1. For the "Promise" version `node promiseVersion.js`
        1. For the "Async/Await" version `node asyncAwaitVersion.js`
    1. When asked, provide the name of the person who is completing the chores.
- **Chore List and Rules** - below is the full list of chores and the time it takes to complete each item. (all items except "Mowing the yard" have a 60% chance of completion)
    1. Mowing the yard - 2 Seconds
    1. Weed eating the yard - 1.5 Seconds
    1. Trim the hedges - 1 Second
    1. Collect wood - 2.5 Seconds
    1. Water the garden - .5 Seconds
