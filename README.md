# Summer Chores
- **Description**: A javascript file that follows someone working through a list of chores.
- **Tool Requirements**: Requires Node.js
- **Starting Program**: Simply:
    1. Run `node callbackVersion.js` in an appropriately scoped terminal (either navigated to the project folder, or an intigrated terminal in the IDE)
    2. When asked, provide the name of the person who is completing the chores.
- **Chore List and Rules** - below is the full list of chores and the time it takes to complete each item.
    1. Mowing the yard - 2 Seconds
    1. Weed eating the yard - 1.5 Seconds
    1. Trim the hedges - 1 Second
    1. Collect wood - 2.5 Seconds
    1. Water the garden - .5 Seconds
